GUIs
====

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   imagechecker/imagechecker
   createjobs/jobcreator

LFD contains 2 GUI interfaces:

* Imagechecker is an image browser developed specifically to help with results
  verification.
* JobCreator is a GUI interface to createjobs module that helps with the creation
  of cluster job scritps because it's capable of displaying all selected parameters
  on-screen in a way that's not possible with the createjobs module.
